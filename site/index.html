隐私政策

最后更新：2026 年 2 月 27 日

概述

悠然标签页（以下简称"本扩展"）是一款开源的浏览器新标签页扩展。我们高度重视您的隐私，本扩展不会收集、存储或传输任何用户个人数据。

数据收集

本扩展不收集任何个人信息或使用数据。所有配置数据（包括站点列表、主题设置、背景设置等）均保存在浏览器本地存储中，不会发送到任何外部服务器。

本地存储

本扩展使用浏览器的本地存储（localStorage / chrome.storage.local）来保存您的个性化设置。这些数据完全存储在您的设备上，您可以随时通过扩展的导出功能备份数据，也可以通过卸载扩展清除所有数据。

网络请求

本扩展仅在获取站点图标时发起网络请求，用于从公共图标服务获取网站的 favicon。除此之外，本扩展不会向任何服务器发送请求。

第三方服务

本扩展不集成任何第三方分析、广告或追踪服务。

权限说明

本扩展仅申请运行所需的最小权限，不请求访问您的浏览历史、书签或其他敏感信息。

政策变更

如本隐私政策有任何变更，我们将在此页面更新。由于本扩展不收集任何数据，预计本政策不会发生重大变化。

开源透明

本扩展完全开源，源代码托管在 GitHub，欢迎任何人审查代码以验证我们的隐私承诺。

联系方式

如有任何隐私相关的问题或疑虑，请通过 GitHub Issues 与我们联系。

使用手册

安装

从 Chrome 应用商店、Edge 扩展商店或 Firefox 附加组件中搜索"悠然标签页"并安装。安装后打开新标签页即可使用。

主题概览

在设置面板中可选择经典、便当、终端、极简、玻璃、气泡、像素或便签 8 种主题。每种主题都有独立的配置项，切换主题不会丢失其他主题的设置。

经典主题

标准图标网格布局，鼠标悬停时显示毛玻璃卡片背景。可配置项包括：容器宽度（400-1400px）、列数（3-8 列）、圆角、卡片透明度、毛玻璃模糊度、图标大小。列数会根据站点数量自动收缩，避免出现多余的空列。

便当主题

提供两种布局模式：
1. 动态布局（默认）：卡片大小由点击次数决定，使用 Treemap 算法自动排列。点击越多的站点占据越大面积，新添加的站点也有初始面积。页面加载时快照点击次数，点击后不会立即重排。
2. 固定网格：所有卡片等分，按设定列数排列。
可配置项包括：容器宽度、间距（4-32px）、圆角、卡片透明度、毛玻璃模糊度、图标大小、布局模式切换、固定模式列数。

极简主题

纯文字列表风格，无图标无卡片，只显示站点名称链接。该主题无专属配置项。

玻璃主题

毛玻璃药丸卡片横向列表布局。可配置项包括：列数（1-4 列）。搜索栏嵌入主界面，始终显示站点标题。适合搭配图片或渐变背景使用。

气泡主题

圆形气泡悬浮效果，不显示站点标题。提供两种布局模式：
1. 固定排列（默认）：气泡按顺序排列，间距均匀。
2. 随机散布：气泡随机散布在画布上，使用碰撞检测算法保证不重叠。
可配置项包括：布局模式切换、气泡大小（48-120px）。搜索栏以可展开的搜索气泡形式呈现。

像素主题

像素复古风格，菜单列表交互。站点以列表形式展示，始终显示站点标题。搜索栏采用命令行提示符样式。鼠标悬停时有扫描线动画效果。该主题无专属配置项。

便签主题

便利贴公告板风格，采用 CSS 多列瀑布流布局。每张便签有彩色背景和图钉装饰，角度随机倾斜，底部内边距不一产生高矮不同的效果。
亮色模式为软木板背景，暗色模式为深色木板背景。搜索栏以便签卡片形式嵌入，带有手绘虚线输入框。可配置项包括：显示或隐藏站点标题。

搜索功能

搜索框使用浏览器默认搜索引擎进行搜索，尊重用户设置。在终端主题中，使用 s 命令加关键词即可搜索。

背景设置

除终端和极简主题外，其他主题均支持背景设置。提供 5 种背景模式：
1. 无背景：跟随浏览器默认主题背景色，保持视觉一体性。
2. 纯色：最多保存 12 个预设颜色。新增时内置算法会根据当前亮色或暗色模式，自动生成适配的颜色。不满意的预设可随时删除并重新生成。
3. 渐变：与纯色类似，最多 12 个预设渐变方案。新增时同样根据当前模式智能生成，支持删除后重新生成。
4. 图片：最多上传 12 张背景图片，上传时自动压缩。
5. 随机：从以上已保存的纯色、渐变和图片中随机选取。
亮色模式和暗色模式的背景配置相互独立。

暗色模式

支持自动跟随系统、手动亮色和手动暗色 3 种模式。暗色模式下可以为站点配置独立的图标。

导入导出

在设置面板中可将站点配置导出为 JSON 文件，方便备份或迁移到其他设备。导入时选择之前导出的 JSON 文件即可恢复配置，导入会自动去重。终端主题也可以使用 export 和 import 命令完成相同操作。

对齐方式

内容支持顶部、居中、底部 3 种垂直对齐方式。每种主题的对齐方式独立保存，互不影响。

重置设置

可将所有设置重置为默认值。重置后会保留已添加的站点数据和点击计数，仅恢复外观和功能配置到初始状态。
